Initialization

The notifications plugin uses underscore.js to render javascript templates. To use the notification system, the page must include the html templates for the notification plugin, which you'll find in the package (look for a file called notification_templates)

You can optionally initialize the notifications plugin on the page with a parent container and a custom class for sizing like this:

    
    new Notifications({
      container: $("body"),
      bootstrapPositionClass: "span6 offset3"
    });

These are actually the default options, which inject the plugin template at the beginning of the container, with a span6 offset3 class for position and size. You should only initialize the plugin if the defaults don't suit your purpose

To push a notification, just call Notifications.push with the following arguments:

imagePath (optional) string the url to your image path. use it if you want an image to the left of your notification
fillImage (optional) boolean defaults to false. use it if you want to expand the image in the whole assigned space, otherwise it will stay centered
text string the message you want inside the notification. can be a html snippet too!
time (optional) string can be used as a timestamp below the text
autoDismiss (optional) integer number of seconds after the notification automatically dismisses itself. if a dismiss count is not provided, the notification must be manually closed by the user
dismiss (optional) function a function that gets called after the notification is dismissed

Buttons

To style something as a button, use class="button"

Optional classes are for colors, and you get by default default blue red green black gray

Other classes you can apply: rounded mini rounded mini rounded inset disabled

All button colors are generated via scss mixins. You can basically create a new button color by using a single color

Dropdowns

Dropdown menus have exactly the same syntax as the ones in the original bootstrap. The difference is of course that instead of btn classes, you use the ones above. For example:

    
    <div class="btn-group">
        <button class="button">Action</button>
        <button class="button dropdown-toggle" data-toggle="dropdown">
          <span class="caret"></span>
        </button>
        <ul class="dropdown-menu">
          <li><a href="#">Action</a></li>
          <li><a href="#">Another action</a></li>
          <li><a href="#">Something else here</a></li>
          <li class="divider"></li>
          <li><a href="#">Separated link</a></li>
        </ul>
    </div>

You can use class dropdown-left on a button toggle to make it open in the left side of the toggle (right aligned)

FAQ page

This is a new html/coffeescript based component for displaying a FAQ page.

Features a live search box which can filter the questions/answers

Basically, just add a ol with some identifier, and the li elements should have a h4 for a question and a p element for the answer

You can add as many questions/answers you like

    
  <ol class="faq-list">
    <li>
      <h4>How do we work?</h4>
      <p>Just fill out our briefing form. This will only take about 5 minutes. Based on that basic information, we will create a design sample in a way we feel best suits your company's focus and vision. You need not waste anymore time on meetings.</p>
    </li>
    ...
  </ol>

After the document is loaded you can call new Faq($(".faq-list")) and it will magically transform in a FAQ page
